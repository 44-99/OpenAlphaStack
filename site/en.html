WHY OPENALPHASTACK

Less agent theater, more verifiable boundaries

An installable foundation for developers exploring China's A-share market—not an automated-profit black box.

01

Domain Skills

Market analysis, screening, stock analysis, and T0 research are composed by one Codex task by default.

02

Typed local MCP

Bounded tools expose market data, indicators, news, backtests, and paper plans—never arbitrary shell or live orders.

03

Deterministic engine

Python enforces T+1, fees, idempotency, state transitions, and the audit ledger. Model prose is metadata, not permission.

THREE CLEAR STAGES

Research → Execution → Evaluation

  1. ResearchCodex composes Skills and cited MCP data.
  2. ExecutionOne publish_paper_plan call enters deterministic checks.
  3. EvaluationSQLite, append-only projections, and the Dashboard preserve evidence.

LOCAL QUICK START

Run it on your machine

Requires Python 3.10+, Node.js 20+, and Codex Desktop.

git clone https://github.com/44-99/OpenAlphaStack.git
cd OpenAlphaStack
pip install -e ".[all]"
npm install
npm run dashboard:build
openalphastack doctor
openalphastack app start

Open http://127.0.0.1:8800/dashboard, then invoke $market-analyzer from Codex. See the English README for the concise evaluation guide.
