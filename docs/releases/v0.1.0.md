# OpenAlphaStack v0.1.0

`v0.1.0` 是 OpenAlphaStack 的第一个公开版本：一个面向 A 股研究、回测与模拟交易
的 Codex 插件栈。它把 Agent 研究工作流放在 Skills，把有类型的本地能力放在
MCP，把交易规则、状态和执行边界留给确定性的 Python。

> 仅用于研究、回测与模拟交易；不连接真实券商，不承诺任何投资收益。

## 本版能力

- `$market-analyzer`、`$stock-screener`、`$stock-analyzer`、`$t0-intraday` 四个领域
  Skills，默认由一个 Codex Agent 按需组合。
- 18 个 MCP 工具，统一返回 `openalphastack.mcp/v1` 信封并保留来源、截至时间、
  新鲜度与 Demo 标记。
- 可离线复现的合成 Demo，覆盖市场概览、新闻、筛选、行情、技术面、基本面和
  基准回测。
- paper/backtest-only 引擎：T+1、费用、风险校验、幂等发布、SQLite 事务状态和
  追加式审计记录。
- 本地 Dashboard：股票搜索、K 线、计划、持仓、账本和三阶段工作流观察。
- `doctor`、Python/前端测试、Dashboard 生产构建与真实 stdio MCP 冒烟验证。

## 快速开始

```powershell
git clone https://github.com/44-99/OpenAlphaStack.git
cd OpenAlphaStack
py -m venv .venv
.\.venv\Scripts\Activate.ps1
python -m pip install -e .
openalphastack doctor
```

完整的插件安装、离线 Skill 提示词、预期输出、Dashboard 和排错步骤见
[首次使用指南](../getting-started.md)。

## 已知限制

- 仅支持研究、回测和模拟交易，不支持实盘。
- Dashboard 没有认证、TLS 或 CSRF 防护，只应监听本机。
- 实时行情依赖第三方数据源，可能不可用或 stale；离线 Demo 必须明确标记为合成
  数据。
- 回测缺少分钟数据时会失败，不会从日线 OHLC 伪造分钟 K 线。
- 调度属于 Codex Desktop，本机和 Codex 必须保持运行；GitHub Actions 不会唤醒
  本地 MCP、引擎或 Dashboard。

## 升级与兼容性

这是首个版本，无历史迁移步骤。需要 Dashboard 的用户请使用 Node.js 20.19 或
更高版本。运行数据保留在本地 `data/`，不要提交到仓库。
