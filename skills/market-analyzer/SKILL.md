# 市场研判流水线

当用户询问市场相关问题时，按以下流水线执行。

## 分析流水线

### 阶段 1：数据获取

```
python -m alphaclaude.tools.quote market                     # 大盘概况：指数/涨跌比/成交额
python -m alphaclaude.tools.flow north                       # 北向资金净流入/5日趋势
python -m alphaclaude.tools.flow {sector_or_code}            # 板块资金流向（按需）
python -m alphaclaude.tools.news market                      # 市场头条新闻
```

### 阶段 2：情绪周期判定

加载 `references/sentiment-cycle.md`，基于换手率、量价结构、新闻面判断当前情绪阶段：

- 冷淡底部 → 逆情绪布局机会
- 平稳 → 正常交易
- 升温介入 → 可适度参与
- 过热警惕 → 减仓信号
- 狂热顶部 → 离场信号

### 阶段 3：板块轮动分析

加载 `references/sector-rotation.md`：
- 识别近期涨幅前列板块
- 判断板块轮动持续性（资金是否持续流入）
- 分析板块内个股分化情况

### 阶段 4：龙头识别

如用户关注具体板块或个股，加载 `references/dragon-head.md`：
- 换手率 > 5%、量比 > 1.5 的领涨股
- 对比个股 vs 板块平均涨幅
- 新闻催化确认

### 阶段 5：输出

1. **市场环境**：大盘趋势/情绪阶段/资金面
2. **板块研判**：强势板块/轮动方向
3. **龙头标的**（如适用）：代码/名称/龙头确认依据
4. **操作建议**：当前市场适合的仓位和策略
