# 龙头策略

**适用市场**: `sector_hot`（板块启动或行业催化剂出现时）
**关联铁律**: #2 趋势交易, #7 强势趋势股放宽
**yaml_name**: dragon_head

核心逻辑：板块轮动中识别领涨股。适用于板块启动或行业催化剂出现时。

---

## 1. 板块领涨地位

从 MCP 市场或候选响应中检查板块排名；若没有板块资金字段，明确标记该项未验证：
- 确认该股所在板块是否为近期涨幅前列
- 确认该股是否在板块启动周期中率先上涨或涨停
- 真正的龙头在上涨日应跑赢板块平均涨幅 2% 以上

## 2. 换手率与动能（关联理念7：强势趋势股放宽）

从 `MCP `stock_quote(code)``：
- 龙头股换手率通常 > 5%
- 量比 > 1.5 说明有活跃的交易兴趣
- 换手率 < 2% 不算龙头

## 3. 相对强度

对比个股涨跌幅与板块平均值：
- 龙头在上涨日跑赢板块 2% 以上 → 确认
- 在下跌日跌幅小于板块均值 → 抗跌性强
- 连续 3 日跑赢板块 → 龙头地位确认

## 4. 新闻催化

从 `MCP `stock_news(code)`` 搜索板块级催化剂：
- 政策利好、行业事件、龙头企业业绩超预期
- 龙头行情常伴随板块整体催化
- 无催化剂的放量上涨 → 可能是游资短炒，持续性存疑

## 5. 乖离率检查（关联理念1：严进策略）

- 龙头股可适当放宽乖离率至 **7%**（vs 普通股的 5%）
- 超过 10% 仍需谨慎，不宜追高
- 放宽不等于无脑追 — 仍需设止损

## 6. 评分调整

| 条件 | 调整 |
|------|------|
| 确认为龙头股（跑赢板块 2%+ 换手率 5%+ 量比 1.5+） | `sentiment_score +10` |
| 板块正处于主动轮动期 + 有催化剂 | 额外 `+5` |

## 7. 输出要求

- 在 `buy_reason` 中注明"龙头策略"判断结果
- 明确给出龙头确认依据（跑赢幅度、换手率、催化剂）
- 止损设在板块启动日低点或个股 MA10
