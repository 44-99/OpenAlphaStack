# 板块轮动分析

**适用市场**: 所有市场环境（不同环境轮动特征不同）
**工具链**: `flow.py` + `quote.py` + `news.py`

---

## 1. 板块资金流向

从 `python tools/flow.py {sector}` 和 `python tools/flow.py north`：

- **北向资金**：连续流入/流出趋势。北向连续 3 日净流入 → 外资看多；连续流出 → 谨慎
- **主力资金**：近 5 日主力净流入板块排名。持续流入的板块轮动概率更大
- **大单方向**：大单买入 vs 大单卖出比例。大单买入 > 60% → 机构建仓迹象

## 2. 板块热度排序

从市场数据判断：

- 涨幅前 5 板块 → 当前主线
- 连续 3 日涨幅居前 → 持续性主线，可积极参与
- 单日脉冲上涨后回落 → 一日游，不参与

## 3. 轮动节奏判断

| 阶段 | 特征 | 策略 |
|------|------|------|
| **启动期** | 板块低位放量、催化消息出现、龙头率先涨停 | 抢先手，仓位 30-50% |
| **主升期** | 板块内多股涨停、换手率持续高位、资金持续流入 | 持有，加仓龙头 |
| **扩散期** | 从龙头扩散到二线、补涨股启动、龙头开始滞涨 | 减仓二线，持有龙头 |
| **退潮期** | 龙头炸板、资金流出、高换手率见顶回落 | 清仓，等待下一轮 |

## 4. 板块内个股分化

- 龙头 vs 跟风：龙头涨幅通常为跟风股的 2-3 倍
- 分化加大 → 板块行情接近尾声
- 不要追板块内涨幅最大的（可能是游资诱多），优先选择基本面支撑 + 放量适中的

## 5. 输出要求

1. 当前强势板块排名（前 3-5）
2. 资金流入/流出方向
3. 轮动阶段判断
4. 板块内龙头 + 可参与标的
5. 仓位建议
