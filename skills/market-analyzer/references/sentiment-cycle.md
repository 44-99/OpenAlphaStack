# 情绪周期策略

**适用市场**: `sector_hot`（板块轮动活跃期）
**关联铁律**: #1 严进策略, #2 趋势交易, #3 效率优先, #5 风险排查
**yaml_name**: emotion_cycle

核心哲学：市场参与者的情绪在"恐慌→悲观→怀疑→希望→乐观→兴奋→贪婪→狂热"之间循环。聪明钱在恐慌底部布局，在狂热顶部离场。

---

## 1. 换手率分析（情绪热度核心指标）

从 `MCP `stock_quote(code)`` 和 `MCP `stock_technical(code, indicator="all")`` 获取换手率数据：

| 换手率 | 情绪阶段 | 含义 |
|--------|----------|------|
| < 0.5%/日 | 冷淡 | 无人关注，潜在底部区域（贪婪时买入别人的恐慌） |
| 0.5% ~ 2% | 正常 | 交投平稳，情绪平稳 |
| 2% ~ 5% | 活跃 | 市场开始关注，不宜追高 |
| > 5% | 高热度 | 游资/散户涌入，警惕情绪顶 |
| > 10%（日均） | 极度过热 | 通常为短期顶部，强烈卖出信号 |

## 2. 连续换手率走势

从 `technical.py` 计算近 20 日换手率走势：

| 走势 | 含义 | 操作 |
|------|------|------|
| 由高向低 + 成交量萎缩 | 情绪退潮 | 耐心等待，不抄底 |
| 由低向高 + 成交量陡增 | 情绪启动 | 可逐步介入 |
| 突然单日暴量（换手率超过前期 5 倍） | 往往是主力出货 | **警惕，不追** |

## 3. 新闻情绪面分析

从 `MCP `stock_news(code)`` 搜索近期新闻，分析情绪倾向：

| 新闻特征 | 情绪判断 |
|----------|----------|
| "利好兑现、业绩超预期、涨停板、机构推荐"集中 | 情绪可能过热，警惕顶部 |
| "业绩下滑、利空、跌破支撑"集中 | 悲观情绪可能造就底部 |
| 散户论坛/社交媒体情绪极端负面 | **反向指标**，可能接近底部 |

## 4. 均线收缩与波动率

从 `MCP `stock_technical(code, indicator="all")``：

- MA5/MA10/MA20 三线粘合（均线收缩）→ 蓄势，方向待定，情绪冷淡
- 波动率降至低位（ATR 萎缩）→ 情绪极度低迷，**蓄势爆发前兆**

## 5. 情绪底部特征（买入区）

满足以下 **3 项以上**即为情绪底部：

- [ ] 近 20 日换手率处于近一年低位
- [ ] 成交量持续萎缩，低于近 60 日均量 50% 以上
- [ ] 近期新闻以低调、中性或负面为主
- [ ] 股价在 MA20 附近或以下，但未出现恐慌性暴跌
- [ ] 机构持仓稳定或小幅增加（如有数据）

## 6. 情绪顶部特征（减仓区）

满足以下 **3 项以上**即为情绪顶部：

- [ ] 近 5 日换手率 > 近 20 日均值的 2 倍
- [ ] 成交量脉冲式放大（单日暴量）
- [ ] 新闻以利好兑现、机构目标价大幅上调、散户追捧为主
- [ ] 股价偏离 MA5 超过 8%（高乖离率）
- [ ] MACD 出现顶背离

## 7. 评分调整

| 条件 | 调整 |
|------|------|
| 情绪底部特征满足 3 项以上 | `sentiment_score +14` |
| 情绪底部特征满足全部 5 项 | `sentiment_score +20` |
| 情绪顶部特征满足 3 项以上 | `sentiment_score -12` |
| 情绪顶部特征满足全部 5 项 | `sentiment_score -20` |
| 情绪平稳区间 | 不调整基础分 |

## 8. 输出要求

必须明确给出：
1. **当前情绪阶段判断**：冷淡底部 / 平稳 / 升温介入 / 过热警惕 / 狂热顶部
2. **当前换手率**与近一年换手率均值对比
3. 是否满足情绪底部或顶部特征（**逐条列出**满足/不满足的条项）
4. 给出**逆情绪操作建议**（大众恐慌我贪婪，大众贪婪我谨慎）
