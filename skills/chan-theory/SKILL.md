# 缠论

核心框架：分型 → 笔 → 线段 → 中枢 → 趋势

## 分析步骤

### 1. 中枢识别
调用 `python tools/technical.py {code} --all` 获取近 60 日日线数据：
- 识别近期价格的高低点序列
- **震荡中枢**（1 个以上中枢）：连续 3 段走势重叠区间
- **趋势段**（脱离中枢向上/向下）：连续 3 个同级别中枢均向同一方向移动

### 2. 背驰判断（最高优先级信号）
- **顶背驰**：价格创新高但 MACD 红柱面积缩小 → 卖出/减仓
- **底背驰**：价格创新低但 MACD 绿柱面积缩小 → 买入/加仓
- 使用 MACD 数据与价格高低点对比

### 3. 买卖点判定
- **一买**（最强）：下跌趋势中，最后一个中枢出现底背驰
- **二买**：离开下跌中枢后的第一次回调不破中枢高点
- **三买**：中枢震荡后向上突破（不回中枢内）
- 一卖/二卖/三卖为对称结构，方向相反
- 当前价格所处的买卖点级别决定仓位大小

### 4. 级别与仓位
- 日线级别买卖点：可用较重仓位 (30-50%)
- 多级别共振（日线+周线同方向）时信号最强

### 5. 输出要求
- 明确说明当前处于：上涨趋势/下跌趋势/中枢震荡
- 指出是否存在背驰信号及背驰级别
- 给出当前买卖点类型，若无则写"暂无明确买卖点"
- 止损设于前低（买入时）或前高（卖出时）

## 评分调整
- 底背驰 + 一买信号：+15
- 二买/三买共振：+10
- 中枢震荡无明确方向：维持基准
- 顶背驰 / 趋势向下：-15
