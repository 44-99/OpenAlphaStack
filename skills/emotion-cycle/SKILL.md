# 情绪周期

核心哲学：市场情绪在"恐慌→悲观→怀疑→希望→乐观→兴奋→贪婪→狂热"之间循环。聪明钱在恐慌底部布局，在狂热顶部离场。

## 情绪阶段量化指标

### 1. 换手率分析（情绪热度核心指标）
调用 `python tools/quote.py {code}` 和 `python tools/technical.py {code} --all`：
- **换手率 < 0.5%/日**：市场冷淡，潜在底部区域
- **换手率 0.5%~2%**：正常交投，情绪平稳
- **换手率 2%~5%**：活跃，市场开始关注
- **换手率 > 5%**：高热度，游资/散户涌入，警惕情绪顶
- **换手率 > 10%（日均）**：极度过热，通常为短期顶部

### 2. 连续换手率走势
计算近 20 日换手率走势：
- 由高向低（持续降温）+ 成交量萎缩 → 情绪退潮，耐心等待
- 由低向高（加速升温）+ 成交量陡增 → 情绪启动，可介入
- 突然单日暴量（换手率超过前期 5 倍）→ 往往是主力出货

### 3. 新闻情绪面分析
调用 `python tools/news.py {code}` 搜索近期新闻：
- 利好兑现、业绩超预期、涨停板、机构推荐集中 → 情绪可能过热
- 业绩下滑、利空、跌破支撑集中 → 悲观情绪可能造就底部

### 4. 均线收缩与波动率
调用 `python tools/technical.py {code} --all`：
- MA5/MA10/MA20 三线粘合 → 蓄势，方向待定
- ATR 萎缩 → 情绪极度低迷，蓄势爆发前兆

## 情绪底部特征（买入区）
满足以下 3 项以上：
- 近 20 日换手率处于近一年低位
- 成交量持续萎缩，低于近 60 日均量 50% 以上
- 近期新闻以低调、中性或负面为主
- 股价在 MA20 附近或以下，但未出现恐慌性暴跌
- 机构持仓稳定或小幅增加（如有数据）

## 情绪顶部特征（减仓区）
满足以下 3 项以上：
- 近 5 日换手率 > 近 20 日均值的 2 倍
- 成交量脉冲式放大（单日暴量）
- 新闻以利好兑现、目标价大幅上调、散户追捧为主
- 股价偏离 MA5 超过 8%（高乖离率）
- MACD 出现顶背离

## 评分调整
- 情绪底部特征满足 3 项以上：+14
- 情绪底部特征满足全部 5 项：+20
- 情绪顶部特征满足 3 项以上：-12
- 情绪顶部特征满足全部 5 项：-20
