# 缩量回踩

## 入场判定标准

### 1. 前提条件（关联理念2：趋势交易）
调用 `python tools/technical.py {code} --all` 确认：
- 股票必须处于上升趋势（MA5 > MA10 > MA20）

### 2. 回踩检测（关联理念4：买点偏好）
- 价格回踩至 MA5 附近（误差 1% 以内）或 MA10 附近（误差 2% 以内）
- 回调期间成交量 < 5 日均量的 70%（缩量特征）

### 3. 反弹信号（关联理念1：严进策略）
- 当前价格守住均线支撑位
- MA5 乖离率 < 2% → 最佳买入区间

### 4. 确认条件（关联理念5：风险排查）
- 调用 `python tools/news.py {code}` 无利空消息
- 获利比例 50-80% 为健康区间

## 评分调整
- 缩量回踩 MA5：+10
- 缩量回踩 MA10 且量能 < 0.6 倍均量：+8
- 理想买点设在 MA5 水平，次优买点设在 MA10
- 止损设在 MA20 水平
