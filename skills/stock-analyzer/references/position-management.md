# 位置判断 — 箱体震荡战法

**适用市场**: `sideways`（横盘震荡）
**关联铁律**: #1 严进策略, #2 趋势交易, #3 效率优先
**yaml_name**: box_oscillation

核心逻辑：箱体内部价格在阻力位与支撑位之间反复震荡，"贴着支撑买、接近阻力卖"，通过波段操作获取区间收益。

---

## 1. 箱体识别

从 `MCP `stock_technical(code, indicator="all")`` 获取近 60-120 日日线数据：

- **箱体顶部（阻力位）**：近期多次触碰但未有效突破的高点连线。通常为 20-60 日内 3 次以上的高点聚集区域。
- **箱体底部（支撑位）**：近期多次下探但未有效跌破的低点连线。通常为 20-60 日内 3 次以上的低点聚集区域。
- **箱体有效性**：顶部和底部各至少触碰 2-3 次方可确认。触碰次数越多，箱体越可靠。
- 使用 `technical.py` 返回的布林带上轨/下轨和均线数据辅助定位支撑阻力。

## 2. 当前位置判断

从 `MCP `stock_quote(code)`` 获取现价，与箱体边界对比：

| 区域 | 判定条件 | 操作 |
|------|----------|------|
| **箱底区域** | 距支撑 ≤ 5% | 买入/加仓信号，止损设箱底下方 3% |
| **箱中区域** | 箱体中间 1/3 | 观望，不主动操作 |
| **箱顶区域** | 距阻力 ≤ 5% | 减仓/止盈信号，不追高 |

## 3. 量能辅助判断

- **箱底放量企稳**：支撑有效的强信号，可较重仓（量比 > 1.2 且收阳）
- **箱顶缩量滞涨**：阻力有效的卖出信号（量比 < 0.8 且收阴/十字星）
- **箱体突破（放量超过均量 2 倍以上）**：
  - 向上有效突破 → 转为多头趋势策略，新目标 = 箱顶 + 箱体高度
  - 向下有效跌破 → 离场等待，原支撑转阻力

## 4. 箱体宽度与预期收益

计算箱体宽度：`(顶部 - 底部) / 底部 × 100%`

| 宽度 | 评估 |
|------|------|
| < 5% | 操作空间过小，不建议参与 |
| 5% ~ 15% | 标准箱体，波段操作可行 |
| > 15% | 大箱体，可做更大波段 |

## 5. 假突破识别

- 单日盘中触及阻力/支撑后快速回撤，收盘回到箱内 → 假突破，维持箱体操作
- 连续两日收盘突破箱体边界，且量能放大（量比 > 1.5）→ 真突破，修改策略

## 6. 评分调整

| 条件 | 调整 |
|------|------|
| 箱底企稳 + 缩量 | `sentiment_score +10` |
| 箱底放量攻顶 | `sentiment_score +12` |
| 箱体向上有效突破（连续两日收盘 + 放量） | `sentiment_score +15`，转趋势策略 |
| 处于箱顶区域 | `sentiment_score -5`（不追高） |
| 箱底有效跌破 | `sentiment_score -15`，离场 |

## 7. 输出要求

必须明确给出：
1. **箱体顶部价位**和**箱体底部价位**
2. **当前价格所处区间**（箱底/箱中/箱顶）
3. 若价格已突破，说明**突破方向**及**新目标价**
4. **建议仓位**与**止损位**
