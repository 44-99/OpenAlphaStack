# 高级分析框架

当常规信号扫描结果模糊（无明确入场信号匹配），或用户明确要求深度分析时使用。
缠论适合震荡和转折市，波浪理论适合趋势市。

---

# 缠论

**适用市场**: `volatile`（高波动/转折市）
**关联铁律**: #1 严进策略, #2 趋势交易, #3 效率优先, #4 买点偏好
**yaml_name**: chan_theory

核心框架：分型 → 笔 → 线段 → 中枢 → 趋势

## 1. 中枢识别

从 `python -m alphaclaude.tools.technical {code} --all` 获取近 60 日日线数据：
- 识别近期价格的高低点序列，判断当前是否在**震荡中枢**（1 个以上中枢）还是**趋势段**（脱离中枢向上/向下）
- 中枢：连续 3 段走势重叠区间，价格在此区间反复震荡
- 趋势：连续 3 个同级别中枢均向同一方向移动

## 2. 背驰判断（最高优先级信号）

- **顶背驰**：价格创新高但 MACD 红柱面积缩小 → 卖出/减仓信号
- **底背驰**：价格创新低但 MACD 绿柱面积缩小 → 买入/加仓信号
- 使用 `technical.py` 获取 MACD 数据（DIF/DEA/柱状图），与价格高低点对比

## 3. 买卖点判定

| 买卖点 | 定义 | 强度 |
|--------|------|------|
| **一买** | 下跌趋势中，最后一个中枢出现底背驰 | 最强买入信号 |
| **二买** | 离开下跌中枢后的第一次回调不破中枢高点 | 确认信号 |
| **三买** | 中枢震荡后向上突破（不回中枢内） | 追涨信号 |
| **一卖** | 上涨趋势中，最后一个中枢出现顶背驰 | 最强卖出信号 |
| **二卖/三卖** | 对称结构，方向相反 | |

当前价格所处的买卖点级别决定仓位大小。

## 4. 级别与仓位

- **日线级别买卖点**：可用较重仓位（30-50%）
- **周线级别买卖点**：可用较大仓位（50-80%）
- **多级别共振**（日线+周线同方向）时信号最强，可满仓

## 5. 评分调整

| 条件 | 调整 |
|------|------|
| 底背驰 + 一买信号 | `sentiment_score +15` |
| 二买/三买共振 | `sentiment_score +10` |
| 中枢震荡无明确方向 | `sentiment_score` 维持基准 |
| 顶背驰 / 趋势向下 | `sentiment_score -15` |

## 6. 输出要求

必须明确给出：
1. 当前处于：上涨趋势 / 下跌趋势 / 中枢震荡
2. 是否存在背驰信号及背驰级别
3. 当前买卖点类型（一买/二买/三买 等），若无则写"暂无明确买卖点"
4. 止损设于前低（买入时）或前高（卖出时）

# 波浪理论

**适用市场**: `volatile`（高波动转折市）
**关联铁律**: #1 严进策略, #2 趋势交易, #3 效率优先, #4 买点偏好
**yaml_name**: wave_theory

核心原则：市场按照 5 浪推进 + 3 浪调整的循环结构运行。

## 1. 识别当前浪型

从 `python -m alphaclaude.tools.technical {code} --all` 获取近 120 日日线数据：

**推动浪（1-3-5）识别特征**：
- 第 1 浪：趋势反转的第一波，成交量温和放大
- 第 3 浪：最强劲的推动浪，通常放大量，MACD 强势；**绝不是最短浪**（波浪铁律）
- 第 5 浪：量能往往弱于第 3 浪，出现顶背离则走高后即将结束

**调整浪（A-B-C）识别特征**：
- A 浪：第一次下跌，成交量较大，多数人以为是回调
- B 浪：反弹，力度弱于前期涨幅，成交量萎缩，**陷阱风险高**
- C 浪：第二次下跌，力度往往超过 A 浪，完成调整

## 2. 斐波那契位置判断

- 第 2 浪回调通常在第 1 浪的 38.2% ~ 61.8%
- 第 3 浪目标通常是第 1 浪的 1.618 ~ 2.618 倍延伸
- 第 4 浪不得进入第 1 浪价格区域（**波浪铁律 — 违反则重新计数**）
- C 浪目标：A 浪顶端起算，等于或超过 A 浪长度

## 3. 最优买点

| 买点 | 时机 | 安全性 | 止损 |
|------|------|--------|------|
| 第 2 浪回调企稳（黄金坑） | 回调至 38.2-61.8% 后企稳 | 最安全 | 第 1 浪起点 |
| 第 3 浪初期突破 | 放量突破第 1 浪高点 | 强势 | 第 1 浪顶部 |
| 第 4 浪回调企稳 | 回调企稳不破第 1 浪高点 | 次优 | 第 1 浪顶部 |

**避免**：第 5 浪末端追高（顶背离风险高）

## 4. 风险提示

- B 浪反弹不宜重仓（陷阱性质，可能为下跌中继）
- 波浪计数存在主观性，需结合其他技术指标验证
- 若波浪规则被违反（如第 4 浪侵入第 1 浪），**需要重新计数**（使用"归数"这一正式术语）

## 5. 评分调整

| 条件 | 调整 |
|------|------|
| 第 2 浪底部企稳（黄金坑） | `sentiment_score +15` |
| 第 3 浪突破确认 | `sentiment_score +12` |
| 第 5 浪末端 / 顶背离 | `sentiment_score -10` |
| C 浪下跌中 | `sentiment_score -12` |

## 6. 输出要求

必须明确给出：
1. 当前可能的浪型位置（如"处于第 3 浪中段"或"疑似第 4 浪调整"）
2. 关键斐波那契支撑/阻力位（0.382 / 0.618 / 1.618）
3. 当前是买入时机、等待机会还是规避
4. 波浪计数的**置信度**（高/中/低）
