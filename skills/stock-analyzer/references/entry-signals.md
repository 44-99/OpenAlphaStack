# 入场信号检查清单

基于已获取的行情和技术指标数据，按趋势判定筛选适用信号，逐项扫描匹配。

---

## 1. 均线金叉

**适用市场**: `trending_up`（不适用于下跌趋势中的金叉）
**关联铁律**: #1 严进策略, #2 趋势交易, #3 效率优先
**yaml_name**: ma_golden_cross

### 判定条件

**主信号 — MA5 上穿 MA10**：
从 `python -m alphaclaude.tools.technical {code} --all` 返回的均线数据中检查：
- MA5 在最近 3 个交易日内上穿 MA10（前一日 MA5 < MA10，当日 MA5 > MA10）
- MA5 当前值 > MA10 当前值

**强信号 — MA10 上穿 MA20**：
- MA10 在最近 5 个交易日内上穿 MA20
- 更慢但可靠性更高

**MACD 确认**：
从 `technical.py --all` 返回的 MACD 数据中检查：
- MACD 金叉（DIF 上穿 DEA）
- 若为零轴上方金叉，信号更强

**量能确认**：
- 金叉日成交量高于 5 日均量
- 从 `python -m alphaclaude.tools.quote {code}` → `volume_ratio` > 1.2 为积极信号

**趋势背景分级**：
- 盘整后金叉：最强信号 ⬆⬆
- 上升趋势中金叉：延续信号 ⬆
- 深度下跌中金叉：弱信号，需更多确认 ⬇

**价格位置**：
- 乖离率 < 5%，避免追高延迟入场
- 价格应在交叉均线附近或上方

### 评分调整

| 条件 | 调整 |
|------|------|
| MA5 × MA10 金叉配合量能 | `sentiment_score +10` |
| MA10 × MA20 金叉 | `sentiment_score +8` |
| MACD 零轴上方金叉 | 额外 `+5` |

### 买卖点

- **理想买点**：交叉均线水平附近
- **止损**：金叉前低点下方 2%
- **输出字段**：在 `ma_analysis` 和 `buy_reason` 中注明"均线金叉"

## 2. 放量突破

**适用市场**: `trending_up`
**关联铁律**: #1 严进策略, #2 趋势交易, #3 效率优先
**yaml_name**: volume_breakout

### 判定条件

**阻力位识别**：
从 `python -m alphaclaude.tools.technical {code} --all` 返回数据中识别：
- 20 日高点
- 布林带上轨
- 前期震荡平台顶部（20-60 日内 3 次以上触碰未突破的高点）

**量能确认**：
- 当日成交量 > 5 日均量的 **2 倍**
- 从 `python -m alphaclaude.tools.quote {code}` → `volume_ratio` > 2.0
- 使用 `technical.py` 返回的日线数据计算均量进行交叉验证

**价格确认**：
- 收盘价必须站上阻力位
- 收盘应在当日振幅上方 30%（强势收盘，即收盘价接近最高价）
- 突破后乖离率检查：仍需 < 5%，避免追高

**后续验证（区分真突破与假突破）**：
- 次日开盘在突破位之上 → 真突破，确认加仓
- 次日回落突破位之下 → 假突破，观望

**风险过滤**：
- `python -m alphaclaude.tools.news {code}` 检查无重大利空
- PE 不应明显高于行业（`python -m alphaclaude.tools.fundamental {code}` 确认）
- `python -m alphaclaude.tools.flow {code}` 确认主力资金方向（主力净流入 → 加分）

### 评分调整

| 条件 | 调整 |
|------|------|
| 放量突破确认（量比>2.0 + 强势收盘 + 站上阻力） | `sentiment_score +12` |
| 突破伴随板块共振（板块也走强） | 额外 `+5` |

### 买卖点

- **理想买点**：突破位附近（突破后回踩确认时）
- **止损**：突破位下方 3%
- **止盈**：突破位 + (阻力位 - 支撑位) 的涨幅延伸
- **输出字段**：在 `buy_reason` 和 `volume_analysis` 中注明"放量突破"

## 3. 缩量回踩

**适用市场**: `trending_up`（主要）, `trending_down`, `sideways`
**关联铁律**: #1 严进策略, #2 趋势交易, #4 买点偏好
**yaml_name**: shrink_pullback

### 判定条件

**前提条件 — 多头排列（必须）**：
从 `python -m alphaclaude.tools.technical {code} --all` 确认：
- MA5 > MA10 > MA20（上升趋势）
- 不满足此条件则不适用此信号

**回踩检测**：
- 价格回踩至 MA5 附近（误差 1% 以内）：最佳
- 或价格回踩至 MA10 附近（误差 2% 以内）：次优
- 回调期间成交量 < 5 日均量的 70%（缩量特征）
- `technical.py` 返回数据中 volume 应显示缩量

**反弹信号**：
- 当前价格守住均线支撑位（未有效跌破）
- 从 `python -m alphaclaude.tools.quote {code}` → 现价 >= MA5 或现价 >= MA10
- MA5 乖离率 < 2% → 最佳买入区间

**确认条件**：
- `python -m alphaclaude.tools.news {code}` 无利空消息
- 获利比例 50-80% 为健康筹码区间

### 评分调整

| 条件 | 调整 |
|------|------|
| 缩量回踩 MA5 + 量能 < 0.7 倍均量 | `sentiment_score +10` |
| 缩量回踩 MA10 且量能 < 0.6 倍均量 | `sentiment_score +8` |

### 买卖点

- **理想买点**：MA5 水平（最佳）
- **次优买点**：MA10 水平
- **止损**：MA20 水平
- **输出字段**：在 `buy_reason` 中注明"缩量回踩"

## 4. 底部放量

**适用市场**: `trending_down`
**关联铁律**: #2 趋势交易, #5 风险排查
**yaml_name**: bottom_volume
**category**: reversal（反转信号，风险高于趋势跟踪）

### 判定条件

**持续下跌确认**：
从 `python -m alphaclaude.tools.technical {code} --all` 获取 30 日日线数据：
- 股价从 20 日高点到近期低点跌幅 > 15%
- 趋势状态应为 BEAR 或 STRONG_BEAR

**量能异动**：
- 当日成交量 > 5 日均量的 **3 倍**
- 从 `python -m alphaclaude.tools.quote {code}` → `volume_ratio` > 3.0
- 该异动应出现在前期极度缩量之后（前 5-10 日成交量低迷）

**价格企稳**：
- 当日 K 线收阳（收盘价 > 开盘价）
- 价格守住近期低点（未创新低）
- 最好出现长下影线（下影线长度 > 实体长度），显示买方支撑

**确认因素**：
- `python -m alphaclaude.tools.news {code}` 确认是否有基本面催化（业绩、政策、行业事件）
- `python -m alphaclaude.tools.fundamental {code}` 配合估值判断（低估值 + 底部放量 = 更强信号）
- 筹码分布：平均成本接近现价（成本收敛）

### 风险提示

- 这是反转信号，风险高于趋势跟踪
- 仓位建议较小（最多 2-3 成）
- 止损必须严格（设在近期低点下方 3%）
- 可能为下跌中继而非真正底部

### 评分调整

| 条件 | 调整 |
|------|------|
| 底部放量确认（量比>3.0 + 收阳 + 守住低点） | `sentiment_score +8` |
| 配合阳线 + 新闻催化 + 低估值 | 额外 `+5` |

### 买卖点

- **买点**：放量阳线次日开盘价（等确认不破低点）
- **止损**：近期低点下方 3%
- **输出字段**：在 `buy_reason` 和 `pattern_analysis` 中注明"底部放量"

## 5. 一阳三阴

**适用市场**: `trending_up`（默认不限制，但趋势多头时可靠性更高）
**关联铁律**: #2 趋势交易, #4 买点偏好
**yaml_name**: one_yang_three_yin

### 形态定义（最近 5 个交易日）

从 `python -m alphaclaude.tools.technical {code} --all` 获取最近 10 日 K 线数据（OHLC）：

1. **第 1 日 — 大阳线**：
   - 收盘价 > 开盘价
   - 实体 > 股价的 2%（(收盘-开盘)/开盘 > 2%）

2. **第 2-4 日 — 三根小 K 线**：
   - 连续三根阴线或小 K 线（每根实体 < 1.5%）
   - 每根 K 线最低价不跌破第 1 日开盘价
   - 成交量应逐步萎缩（每日量比 < 0.8）
   - 三根 K 线应收在第 1 日实体范围内

3. **第 5 日 — 突破阳线**：
   - 又一根阳线（收盘 > 开盘）
   - 收盘价突破第 1 日收盘价

### 趋势确认

- 多头排列确认（MA5 > MA10 > MA20），趋势向上 → 信号可靠
- 趋势不明朗（均线缠绕）→ 信号降级

### 评分调整

| 条件 | 调整 |
|------|------|
| 形态成立 + 趋势看多（多头排列） | `sentiment_score +15` |
| 形态成立但趋势不明（均线缠绕） | `sentiment_score +5` |

### 买卖点

- **理想买点**：第 5 日收盘价附近
- **止损**：第 1 日开盘价下方 2%
- **输出字段**：在 `pattern_analysis` 和 `buy_reason` 中注明"一阳夹三阴"
